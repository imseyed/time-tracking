# Time Tracking Panel

## قابلیت‌های اصلی
- ثبت ساعت کاری با تاریخ **شمسی**
- نمایش **25 رکورد اخیر کاربر** در صفحه ثبت ساعت
- گزارش‌گیری با پیش‌فرض **30 روز اخیر**
- نمودار میله‌ای **عمودی** ساعت روزانه
- مدیریت کاربران (لیست + ایجاد/ویرایش/حذف)
- مدیریت پروژه‌ها (لیست + ایجاد/ویرایش/حذف)
- منو و دکمه‌ها با آیکن

## دسترسی‌ها
- ادمین: `admin / public`
- ادمین: ایجاد/ویرایش/حذف کاربر، ایجاد/ویرایش/حذف پروژه، گزارش‌گیری همه
- کاربر عادی: ثبت ساعت و گزارش خودش

## فونت‌ها
فایل فونت‌ها را در مسیر زیر قرار دهید:

- `frontend/public/fonts/IRANSansXFaNum-Medium.woff2`
- `frontend/public/fonts/IRANSansXFaNum-ExtraBold.woff`

## Backend

```bash
cd backend

```
