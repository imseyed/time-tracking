# Time Tracking Panel

## قابلیت‌های اصلی
- ثبت ساعت کاری با تاریخ **شمسی**
- نمایش **25 رکورد اخیر کاربر** در صفحه ثبت ساعت
- گزارش‌گیری با پیش‌فرض **30 روز اخیر**
- نمودار میله‌ای **عمودی** ساعت روزانه
- مدیریت کاربران (لیست + ایجاد/ویرایش/حذف)
- مدیریت پروژه‌ها (لیست + ایجاد/ویرایش/حذف)
- منو و دکمه‌ها با آیکن

## دسترسی‌ها
- ادمین: `admin / public`
- ادمین: ایجاد/ویرایش/حذف کاربر، ایجاد/ویرایش/حذف پروژه، گزارش‌گیری همه
- کاربر عادی: ثبت ساعت و گزارش خودش

## فونت‌ها
فایل فونت‌ها را در مسیر زیر قرار دهید:

- `frontend/public/fonts/IRANSansXFaNum-Medium.woff2`
- `frontend/public/fonts/IRANSansXFaNum-ExtraBold.woff`

## معرفی پروژه
این پروژه یک سامانه ثبت و گزارش‌گیری ساعت کاری (Time Tracking) برای تیم‌ها است که با رابط فارسی و تاریخ شمسی طراحی شده است.

- معماری پروژه به‌صورت دو بخش مجزا است:
- `frontend`: اپلیکیشن تک‌صفحه‌ای با `Svelte + Vite`
- `backend`: API سبک مبتنی بر `PHP (PDO)` با پایگاه داده `MySQL`

- منطق احراز هویت و سطح دسترسی بر اساس نقش کاربر پیاده‌سازی شده است:
- `admin`: مدیریت کاربران، مدیریت پروژه‌ها، ثبت ساعت برای کاربران، گزارش‌گیری کامل
- `user`: ثبت ساعت و مشاهده گزارش شخصی

- جریان اصلی داده در سیستم:
- کاربر در فرانت‌اند وارد می‌شود و شناسه کاربر به API ارسال می‌گردد.
- ثبت ساعت با تاریخ شمسی انجام می‌شود و در فرانت‌اند به تاریخ میلادی تبدیل شده و در دیتابیس ذخیره می‌شود.
- گزارش‌ها با فیلتر کاربر/پروژه/بازه زمانی تولید شده و شامل جزئیات رکوردها، تجمیع روزانه و خلاصه آماری هستند.

- مدل داده در دیتابیس شامل سه جدول اصلی است:
- `users`: کاربران و نقش‌ها
- `projects`: پروژه‌ها و رنگ نمایشی
- `time_entries`: رکوردهای زمانی شامل تاریخ، بازه زمانی، شرح کار و مدت (دقیقه)
